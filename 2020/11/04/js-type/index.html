JavaScript主要内容（一）

JavaScript
前端
发布日期:   2020-11-04

JavaScript是一种多范例的动态语言，具有类型和运算符，标准的内置对象和方法。

JavaScript的类型主要为Number、String、Boolean、Function、Object、Symbol，以及undefined、null和一些内置的Error类型。其中Object中有一些特殊的对象，如Array、Date、RegExp。

下面将介绍JavaScript中部分主要类型：

Numbers

精度

在JavaScript中，数字为64位双精度形式，也就是说，没有整型数据（除了BigInt）。因此，有些数据看上去是整型，但事实上是一个浮点数。在实际中，整型数会被当成32位整型，一些实现甚至会以这种方式进行存储，直到需要实现一条对Number有效的指令，而不是对32位整型有效的指令。这对于位操作很重要。

console.log(3 / 2);             //1.5
console.log(Math.floor(3 / 2)); //1
0.1 + 0.2 == 0.30000000000000004;

算数操作&Math

Number支持标准的算数操作，包括加减模（或取余）算数等。之前用到的Math是一个内置对象，它提供更高级的算术函数和常量：

Math.sin(3.5);
var circumference = 2 * Math.PI * r;

parseInt()&数字解析

内置函数parseInt()可以将字符串解析为整数，函数的第二个参数即为转换的基数。在一些旧的浏览器中，“0”开头的字符串会被认为是8进制的数，不过目前已经不使用了。但是“0x”开头的字符串现在还是会被认为是16进制的数。

parseInt('123', 10);    // 123
parseInt('010', 10);    // 10
parseInt('11', 2);      // 3
parseInt('010');        // 8  目前不使用了
parseInt('0x10');       // 16

类似地，parseFloat()可以将字符串转换为浮点数，但这个函数使用的一直是十进制。

“+”操作符也可以用来把字符串转换成数字：

+ '42';     // 42
+ '010';    // 10
+ '0X10';   // 16

注： parseInt()和parseFloat()函数会一直解析字符串直到遇到一个无效的数字形式的字符，然后返回之前解析得到的数字。“+”操作符如果遇到字符串中有无效字符，则是简单地转化成NaN。

parseInt('10.2abc');    // 10
parseFloat('10.2abc');  // 10.2
+ '10.2abc';            // NaN

特殊值： NaN、Infinity、 -Infinity

在parseInt()中，如果字符串不是数字，则会返回一个特殊值NaN（Not a Number）。

parseInt('hello', 10);  // NaN

如果将其作为任何算术运算的操作数，结果都是NaN。

NaN + 5;    // NaN

也可以使用内置函数isNaN()测试是否为NaN。

isNaN(NaN); // true

Infinity和-Infinity和NaN可以使用内置函数isFinite()测试：

1 / 0;  // Infinity
-1 / 0; // -Infinity
isFinite(1 / 0);    // false
isFinite(-Infinity);    // false
isFinite(NaN);      // false

Strings

JavaScript中的字符串是Unicode字符的序列。准确而言，它们是UTF-16 code unit的序列，每个code unit由一个16位数字表示，每个Unicode字符由1或2个code unit表示。

如果要表示单个字符，只要使用只包含一个字符的字符串即可。

在使用字符串时可以像使用对象，它提供了各种方法（methods）来操作或访问字符串信息。

'hello'.length;     // 5
'hello'.charAt(0);  // "h"
'hello,world'.replace('world', 'mars');     // "hello, mars"
'hello'.toUpperCase();  // "HELLO"

其他类型

null&undefined

在JavaScript中对null和undefined进行了区分。

  • null：特意表示nonvalue，并且只能通过null关键词访问
  • undefined：表示未初始化的变量，事实上undefined是一个常量。

Boolean

布尔类型由两种值：true和false。任何值都可以根据以下规则转化成一个布尔值：

  1. false，0， 空串（””），NaN，null和undefined都变成false;
  2. 其余值都变成true

上述转换可以通过Boolean()函数实现：

Boolean('');    //false
Boolean(234);   //true

布尔类型支持与（&&）或（||）非（！）操作。
