css学习笔记03 —— 选择器

CSS
前端
发布日期:   2020-12-14

选择器

CSS选择器是CSS规则的第一部分。选择器所选择的元素，叫做选择器的对象。

选择器列表

如果有多个元素需要使用同一条CSS规则时，可以把它们结合成一个选择器列表。不同选择器之间用逗号进行分隔，都好前后的空格都有效。

注：如果选择器列表中有一个选择器无效，那么整个规则都会无效。

h1 { 
  color: blue; 
} 

.special { 
  color: blue; 
} 

/* 等价于下方 */
h1, .special { 
  color: blue; 
} 
/* 或者 */
h1, 
.special {
  color: blue; 
}

选择器的类型

  • Type, class, and ID slectors:

    /*Type selector： 以HTML元素为目标*/
h1 { }  
/*class selector*/
.box { }
/*ID selector*/
#unique { }

    • Type selector: 类型选择器也叫标签名选择器(tag name slector)或元素选择器(element selector)。

    • Universal selector: 用*表示，它选择了文档中的所有元素。使用这个也可以使代码更易阅读。

      /*这个选择器很容易和下面的混淆*/
/*第一个选择器希望在<article>的所有后代元素中，找到那些是它们父元素的第一个孩子的元素*/
article :first-child {
  font-weight: bold;
}
/*第二个选择器则是选择了那些是其他元素的第一个子元素的<article>元素*/
article:first-child {
  font-weight: bold;
}
/*为了便于理解，可以把第一个选择器改写成以下形式：*/
article *:first-child { 
  font-weight: bold;
}

    • Class selector: 以’.’号开头。

      /*针对特定元素的类*/
h1.highlight {
  background-color: pink;
}
/*针对具有多个类的元素*/
.notebox {
  border: 4px solid #666;
  padding: .5em;
}

.notebox.warning {
  border-color: orange;
  font-weight: bold;
}
      <div class="notebox">
    This is an informational note.
</div>

<div class="notebox warning">
    This note shows a warning.
</div>
      • ID selector: 以’#’开头，用法和类选择器相似。但是在一个页面中，一个id只能使用一次，且一个元素只能有一个id。

  • Attribute selectors

    /*选择一个元素中的特定属性*/
a[title] { }
/*选择一个带有特殊值的属性*/
a[href="https://example.com"] { }

    • Presence and value selector：

      选择器 举例 说明
      [attr] a[title] 匹配有attr属性的元素
      [attr=value] a[href=”https://example.com"] 匹配attr属性值恰好是value的元素
      [attr~=value] p[class~=”special”] 匹配那些attr属性值恰好是value或者属性值列表中含有value的元素
      [attr =value] div[lang

    • Substring matching selectors：

      选择器 举例 说明
      [attr^=value] li[class^=”box-“] 匹配那些attr属性值以value开头的元素
      [attr$=value] li[class$=”-box”] 匹配那些attr属性值以value结尾的元素
      [attr*=value] li[class*=”box”] 匹配那些attr属性中含有value的元素

    • 大写敏感：如果希望在匹配属性值时大小写不敏感，可以在闭合中括号之前加一个’i’。

      li[class^="a" i] {
    color: red;
}

  • pseudo-classes and pseudo-elements

    /*在元素的某个特定状态下应用样式*/
/* :hover  这个伪类只选择了被鼠标指针hover时的元素*/
a:hover { } 

/*为元素选择器，只选择了某个元素的特定部分*/
/* ::first-line  值会选择元素中第一行文本*/
p::first-line { }

    -

  • Combinators

    /*通过组合一些选择器来确定文档中的目标元素*/
/*以下例子选择了<article>元素中直接子段落元素*/
/* '>' 子运算符*/
article > p { }
